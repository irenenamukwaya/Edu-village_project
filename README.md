## Kids Learning Game

This project is a **Flutter-based educational mobile game for young children** designed to make learning interactive and fun. The application combines several mini-games that help children develop basic cognitive and academic skills through images, sounds, and simple challenges.

The game contains **four mini-games**:

1. **Activity Recognition** – Children are shown images of everyday activities (such as sweeping, reading, or fetching water) and choose the correct description from multiple options.

2. **Object Recognition** – Children identify common objects by selecting the correct name for an image (for example, ball, cup, dog, or book).

3. **Word Pronunciation** – Children learn how to pronounce words using a tap-to-listen feature and read simple example sentences to build vocabulary.

4. **Math and Counting** – Children learn numbers and practice counting using visual objects. The game introduces basic arithmetic such as addition, subtraction, multiplication, and division.

The game includes **reward coins, encouraging feedback, and audio pronunciation** to keep children motivated while they learn. Images, sound effects, and simple interactions are used to create a friendly learning environment suitable for **early learners and primary school children**.

The project is built using **Flutter** and uses local assets, JSON-based question data, text-to-speech for pronunciation, and a simple coin reward system to track progress.
